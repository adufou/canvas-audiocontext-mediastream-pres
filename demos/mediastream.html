MediaStream API

A MediaStream is a collection of tracks. Tracks come from cameras, microphones, canvases, or synthesized sources — and can be mixed freely.

Tracks: video — none audio — none → combined stream — not ready
1

canvas.captureStream()

A live canvas animation can be captured as a MediaStream at a given frame rate. The resulting stream has one VideoStreamTrack — and it's live: every canvas frame appears in the stream.

2

getUserMedia — microphone access

getUserMedia requires HTTPS or localhost. Serve this file with: python3 -m http.server and open http://localhost:8000

navigator.mediaDevices.getUserMedia() returns a Promise that resolves to a MediaStream. We mute the audio element to prevent feedback — but the track is live and usable.

mic — not requested
3

AudioContext → MediaStream

audioCtx.createMediaStreamDestination() creates a node whose .stream is a live MediaStream. The audio is not played to the speakers — only available as a track. Perfect for recording or mixing without feedback.

synth audio track — not created
4

new MediaStream([videoTrack, audioTrack])

new MediaStream([...tracks]) is the compositor. Pick any tracks from any source — camera, canvas, microphone, synthesizer — and combine them into a single stream. This stream can be played, recorded, or sent over WebRTC.

combined stream — not ready
Bonus: MediaRecorder — record the stream to a file

MediaRecorder records any MediaStream to a Blob. Here we record the combined stream for 3 seconds and offer a download.

mediastream.html JS 
// Step 1 — canvas.captureStream()

const canvas = document.getElementById('animCanvas');

// Check support (Safari requires a flag)
if (!canvas.captureStream) {
  console.error('captureStream not supported');
}

// Capture at 30 fps → returns a MediaStream
const stream = canvas.captureStream(30);

// The stream has one VideoStreamTrack
const [videoTrack] = stream.getVideoTracks();
console.log(videoTrack.kind);   // "video"
console.log(videoTrack.label);  // "canvas"

// Play the stream in a video element
const video = document.getElementById('canvasVideo');
video.srcObject = stream;

// To stop: stop all tracks
stream.getTracks().forEach(t => t.stop());
// Step 2 — getUserMedia (microphone)

// Requires HTTPS or localhost
const stream = await navigator.mediaDevices
  .getUserMedia({ audio: true, video: false });

const [audioTrack] = stream.getAudioTracks();
console.log(audioTrack.kind);   // "audio"
console.log(audioTrack.label);  // "Built-in Microphone"

// Hook into AudioContext to visualize the mic level
const source   = audioCtx.createMediaStreamSource(stream);
const analyser = audioCtx.createAnalyser();
source.connect(analyser);
// Note: NOT connected to destination → no feedback

// Read amplitude for the meter
const data = new Float32Array(analyser.fftSize);
analyser.getFloatTimeDomainData(data);
const rms = Math.sqrt(data.reduce((s,v) => s + v*v, 0) / data.length);
// Step 3 — AudioContext → MediaStream

const audioCtx = new AudioContext();

// MediaStreamDestination node — does NOT play to speakers
const dest = audioCtx.createMediaStreamDestination();

// Connect an oscillator into it
const osc = audioCtx.createOscillator();
osc.frequency.value = 440;
osc.connect(dest);
osc.start();

// dest.stream is a live MediaStream
const synthStream = dest.stream;
const [audioTrack] = synthStream.getAudioTracks();

// The oscillator runs silently until we use the track
console.log(audioTrack.kind);  // "audio"
// Step 4 — Combine tracks

// Pick one track from each source
const videoTrack = canvasStream.getVideoTracks()[0];
const audioTrack = synthStream.getAudioTracks()[0];

// new MediaStream([...tracks]) is the compositor
const combined = new MediaStream([videoTrack, audioTrack]);

// Play it back
const video = document.getElementById('combinedVideo');
video.srcObject = combined;

// combined.getTracks() returns [videoTrack, audioTrack]
// You could also add a mic audio track instead of the synth,
// or mix multiple audio tracks via AudioContext and route
// the result to a single MediaStreamDestination.
// Bonus — MediaRecorder

const recorder = new MediaRecorder(combined);
const chunks = [];

recorder.ondataavailable = e => chunks.push(e.data);
recorder.onstop = () => {
  const blob = new Blob(chunks, { type: 'video/webm' });
  const url  = URL.createObjectURL(blob);
  // Create a download link
  link.href     = url;
  link.download = 'recording.webm';
  link.style.display = 'inline-flex';
};

recorder.start();
setTimeout(() => recorder.stop(), 3000);
