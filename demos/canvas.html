Canvas API

The 2D rendering context — drawing images, managing state with a stack, and clipping regions.

1

drawImage

ctx.drawImage() accepts an HTMLImageElement, HTMLCanvasElement, or HTMLVideoElement (which we'll use in the MediaStream demo).

drawImageJS
// drawImage(source, x, y, w, h)
const ctx = canvas.getContext('2d');

const img = new Image();
img.src = '...';
img.onload = () => {
  ctx.drawImage(img, 50, 25, 300, 200);
};

// Also accepts:
// HTMLCanvasElement
// HTMLVideoElement ← see MediaStream demo
// ImageBitmap, OffscreenCanvas
2

save / restore — the state stack

ctx.save() pushes the current state (transforms, styles, clip) onto a stack. ctx.restore() pops it. Transforms applied inside a save/restore block don't leak out.

Stack depth: 0
save / restoreJS
// save() pushes current state:
//  - transform matrix
//  - clipping region
//  - strokeStyle, fillStyle...
ctx.save();        // depth: 1

ctx.translate(100, 50);
ctx.rotate(Math.PI / 6);
ctx.fillRect(0, 0, 80, 40);

ctx.restore();      // depth: 0

// Back to original origin —
// translate + rotate are gone
ctx.fillRect(200, 100, 80, 40);
3

clip — masking with a path

After calling ctx.clip(), all subsequent drawing is masked to the current path. Wrap in save() / restore() to remove the clip.

clipJS
// Save before clipping
ctx.save();

// Define the clip path
ctx.beginPath();
ctx.arc(200, 150, 120, 0, Math.PI * 2);

// Subsequent drawing is masked here
ctx.clip();

ctx.fillStyle = '#7c6af7';
ctx.fillRect(0, 0, 400, 300);

// restore() removes the clip region
ctx.restore();

// This now fills the whole canvas
ctx.fillRect(0, 0, 400, 300);
4

Combined — animated drawImage + clip + save/restore

Putting it all together: an image drawn each frame, clipped to a circle, rotating inside a save()/restore() block.

combinedJS
function draw(img) {
  const cx = canvas.width / 2;
  const cy = canvas.height / 2;
  const r  = 120;

  ctx.clearRect(0, 0, canvas.width, canvas.height);

  // save() — clip + rotate stay contained
  ctx.save();

  // 1. Circular clip path
  ctx.beginPath();
  ctx.arc(cx, cy, r, 0, Math.PI * 2);
  ctx.clip();

  // 2. Move to center, apply rotation
  ctx.translate(cx, cy);
  ctx.rotate(angle);

  // 3. Draw image centered on rotated origin
  ctx.drawImage(img, -r, -r, r * 2, r * 2);

  // restore() — clip and transform gone
  ctx.restore();

  angle += 0.012;
  requestAnimationFrame(() => draw(img));
}
