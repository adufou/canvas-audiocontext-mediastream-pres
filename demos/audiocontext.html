AudioContext + OscillatorNode

Browsers require a user gesture before any audio can play. Click Start AudioContext first — this unlocks the audio system. OscillatorNodes are one-shot: a new node is created on each Play click.

not started
AudioContext + OscillatorJS
// Must be inside a user-gesture handler
const audioCtx = new AudioContext();
await audioCtx.resume();

// Oscillators are ONE-SHOT:
// .start() can only be called once per node.
// Create a new oscillator each time you play.
let osc = audioCtx.createOscillator();
osc.type = 'sine'; // sine|square|triangle|sawtooth
osc.frequency.setValueAtTime(440, audioCtx.currentTime);

osc.connect(audioCtx.destination);
osc.start();

// To stop:
osc.stop();
osc.disconnect();

GainNode — volume control

A GainNode multiplies the signal by its gain.value. Audio parameters can be automated over time with .setTargetAtTime() to avoid clicks and pops.

GainNodeJS
const gain = audioCtx.createGain();
gain.gain.value = 0.7; // 0 = silence, 1 = full

// osc → gain → speakers
osc.connect(gain);
gain.connect(audioCtx.destination);

// Smooth ramp — avoids clicks
gain.gain.setTargetAtTime(
  newVolume,
  audioCtx.currentTime,
  0.01  // time constant (s)
);

// Or direct (fine for small changes):
gain.gain.value = 0.5;
3

BiquadFilterNode — EQ / filtering

Filters shape the frequency spectrum. Try lowpass at 500 Hz with the square wave — you'll hear the harmonics disappear, rounding the harsh tone.

BiquadFilterNodeJS
const filter = audioCtx.createBiquadFilter();
filter.type            = 'lowpass';
filter.frequency.value = 2000; // cutoff in Hz
filter.Q.value         = 1;    // resonance

// osc → gain → filter → speakers
osc.connect(gain);
gain.connect(filter);
filter.connect(audioCtx.destination);

// Update live — values take effect immediately
filter.frequency.value = 500;
filter.type = 'highpass';

AnalyserNode → Canvas — real-time visualizer

The AnalyserNode exposes audio as raw data — time-domain (waveform) or frequency-domain (FFT). Drawn on a Canvas each frame.

AnalyserNodeJS
const analyser = audioCtx.createAnalyser();
analyser.fftSize = 2048; // power of 2

filter.connect(analyser);
analyser.connect(audioCtx.destination);

// FFT spectrum
const freqData = new Uint8Array(analyser.frequencyBinCount);
analyser.getByteFrequencyData(freqData);

// Waveform
const timeData = new Uint8Array(analyser.fftSize);
analyser.getByteTimeDomainData(timeData);

let animating = true;
function drawLoop() {
  if (!animating) return;
  requestAnimationFrame(drawLoop);
  analyser.getByteFrequencyData(freqData);
  // ... draw bars on canvas ...
}
5

Full signal chain

Wire all nodes with chained .connect() calls. Watch the signal flow diagram above light up as audio passes through each node.

Full chainJS
// .connect() returns the destination node
// so calls can be chained:
osc
  .connect(gain)
  .connect(filter)
  .connect(analyser)
  .connect(audioCtx.destination);

osc.start();

// Stop cleanly:
osc.stop();  // fires osc.onended

osc.onended = () => {
  osc.disconnect();
  animating = false; // stops visualizer
};
